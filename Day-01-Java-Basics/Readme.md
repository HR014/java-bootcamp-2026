# What is Programming?

Imagine you have a robot.

The robot only follows instructions.

Programming is simply writing instructions.

Example

```
Wake up
Brush teeth
Eat breakfast
Go to office
```

That is an algorithm.

Programming is writing algorithms in a language the computer understands.

# Variables

A variable stores information.

Example

```java
String name = "Harsh";
```

"name" stores text.

Example

```java
int age = 24;
```

Stores a whole number.

Example

```java
double salary = 50000.75;
```

Stores decimal numbers.

Example

```java
boolean student = true;
```

Stores true or false.
